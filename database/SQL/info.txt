Ce fichier contient des informations sur le schéma de la base de données utilisée dans ce projet.

Le schéma de la base de données est défini dans le fichier "tables.sql". Il comprend toutes les tables nécessaires pour le projet, y compris deux administrateurs :
- Echarkaouy Ayyoub
- Oumaima Chihab

Pour accéder à la page d'un des deux administrateurs, vous pouvez utiliser les informations de connexion suivantes :
- pour Ayyoub : email = 'ayoub@cieva.com', mot de passe = 'Ayoub@123',
- pour Oumaima : email = 'oumaima@cieva.com', mot de passe = 'Oumaima@123'.

Le schéma de la base de données est conçu pour stocker des informations sur les clients, les produits, les commandes et les éléments de commande.
    La table des clients contient des informations sur chaque client, telles que son nom, son adresse e-mail et son mot de passe.
    La table des produits contient des informations sur chaque produit, telles que son nom, sa description et son prix. La table des commandes contient des informations sur chaque commande, telles que le client qui a passé la commande, le coût total de la commande et la date de la commande.
    La table des éléments de commande contient des informations sur chaque élément de commande, telles que le produit commandé, la quantité commandée et le coût total de l'élément.

Le schéma de la base de données comprend également des tables pour stocker des informations sur les catégories de produits et les images de produits.
    La table des catégories contient des informations sur chaque catégorie de produit, telles que son nom et sa description. La table des images contient des informations sur chaque image de produit, telles que son nom de fichier et le produit auquel elle est associée.
